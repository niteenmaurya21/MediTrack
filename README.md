# MediTrack

MediTrack is a console-based Hospital Management System developed using Core Java.

The application demonstrates object-oriented programming concepts by allowing users to manage doctors, patients, appointments, and billing through a menu-driven console interface.

Features

- Doctor Management
  - Add Doctor
  - View Doctors
  - Search Doctor

- Patient Management
  - Add Patient
  - View Patients
  - Search Patient

- Appointment Management
  - Book Appointment
  - View Appointments
  - Search Appointments
  - Cancel Appointments

- Billing
  - Bill Generation
  - Bill Summary

Technologies Used

- Java
- OOP
- Collections Framework (ArrayList, HashMap)
- Generics
- Exception Handling
- Enums
- File I/O (if implemented)
- Git & GitHub
